# FlickrBrowserApplication
Android Application to Browse Images from Flickr
 
**Demo:**  
[![Flickr Application](https://user-images.githubusercontent.com/90371913/152651510-a77abe2a-38dd-4486-a703-a2d0717d6f2b.jpg)](https://youtu.be/yaXrvIMjLdY)  


Using **Kotlin**  

This Code Provide Simple **Android Application** For **Browsing Images**

This Application Called **Flickr Browser Application**

This Application Uses **https://www.flickr.com/services/api/** API  
This Application Uses **SQLite**
&nbsp;  
&nbsp;  

**This Application Will:**
* • Search Images from Flickr API
* •	Get Images from Flickr API
* •	Browse Images as List
* •	Browse Images as Collection
* •	Display Full Screen Images
* • Save Favorite Images in SQLite Database
* • Browse Favorite Images from SQLite Database
* • Display Full Screen Favorite Images from SQLite Database
* • Delete Images from SQLite Database
